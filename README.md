# PlanOut

- Flask-based 
- Google MySQL Database
- React Application
- Cloud Run


Sources:

- [MySQL Documentation](https://dev.mysql.com/doc/refman/5.7/en/)
- [Connecting Cloud Run to Cloud SQL](https://cloud.google.com/sql/docs/mysql/connect-instance-cloud-run#console)
